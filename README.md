# FileZen GitHub Pages

这个目录是为 GitHub Pages 准备的纯静态站点，无需构建步骤。

## 文件说明

- `index.html`：产品首页
- `privacy.html`：隐私政策
- `terms.html`：使用条款
- `styles.css`：统一样式
- `script.js`：少量页面脚本

## 发布方式

1. 将 GitHub Pages 的发布来源指向当前仓库的 `docs` 目录。
2. 发布后，首页通常对应仓库站点根路径。
3. 如需把应用内隐私链接直接指向隐私政策页，可将 App 的隐私政策 URL 配置为 `.../privacy.html`。
